About

Network Engineer

Saya adalah seorang Network Engineer berpengalaman dengan spesialisasi dalam desain, implementasi, dan manajemen jaringan berbasis teknologi Mikrotik, ZTE, dan Cisco. Dengan latar belakang yang kuat di bidang jaringan, saya memiliki kemampuan untuk merancang solusi infrastruktur yang andal, aman, dan efisien untuk berbagai kebutuhan bisnis.

Selain keahlian dalam perangkat jaringan, saya juga berpengalaman dalam mengelola dan mengimplementasikan virtualisasi server menggunakan Proxmox. Kemampuan ini memungkinkan saya untuk membangun lingkungan server yang fleksibel dan hemat sumber daya bagi perusahaan atau proyek.

Untuk melengkapi keahlian teknis, saya juga seorang Web Application Developer yang mampu merancang dan mengembangkan aplikasi web dinamis sesuai kebutuhan pengguna. Dengan pemahaman mendalam tentang pengembangan aplikasi berbasis web, saya mengintegrasikan solusi jaringan yang efektif dengan platform digital untuk meningkatkan produktivitas dan efisiensi.
